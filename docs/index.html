Pinocchio

Claude's nose stops growing.

Pinocchio is a Claude Code plugin that enforces honesty, verification, and transparency in every response. Same plugin shape as Caveman Mode, opposite axis — brevity vs honesty.

Install

claude plugin marketplace add cooldadpresident/pinocchio
claude plugin install pinocchio

What it catches

🤥 Without Pinocchio

"I've updated the auth middleware and it should work now."

✓ With Pinocchio

"Edited auth.ts:42 — replaced < with <=. [UNVERIFIED] — tests not run."

🤥 Without Pinocchio

"Great question! The function probably returns null on empty input."

✓ With Pinocchio

"getUser() returns null on empty input — verified at src/user.ts:23."

Levels

LevelBehavior
liteFlag hedges inline with 🤥. Don't block.
fullDefault. Block hedge phrases. Require [UNVERIFIED] tag.
strictEvery factual claim needs file:line / URL / command output.

Commands

CommandEffect
/pinocchioToggle and report status
/pinocchio-review [PR#]Flag claim-vs-diff mismatches
/pinocchio-commitHonest commit message from staged diff
stop pinocchioDisable for session
